# Data_Cleaning_Layoffs
This project focuses on cleaning and preprocessing a global layoffs dataset using MySQL.
The goal is to transform raw, inconsistent data into a clean, structured, and analysis-ready table by handling duplicates, standardizing text fields, fixing date formats, and removing invalid records.

This project is ideal for:

Practicing SQL data cleaning

Preparing datasets for EDA or visualization

Demonstrating real-world data preprocessing skills
